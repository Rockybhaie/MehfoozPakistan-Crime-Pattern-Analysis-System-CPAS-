MehfoozPakistan (CPAS) is a data-driven crime management and analysis system designed to modernize crime reporting, investigation tracking, and predictive policing in Pakistan. The system provides a centralized platform for managing crimes, investigations, suspects, victims, witnesses, and evidence while enabling analytical insights and crime pattern detection.

Key Features

Centralized crime and investigation management
Role-based access control (Officers, Victims, Witnesses)
Evidence chain-of-custody tracking
Crime analytics, hotspot identification, and trend analysis
Predictive risk assessment using historical crime data

Tech Stack

Frontend: React.js
Backend: Node.js (Express)
Database: Oracle Database (Triggers, Procedures, Views)
Authentication: JWT & bcrypt
